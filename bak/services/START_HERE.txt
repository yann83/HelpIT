Étape 1 : Copier les 3 fichiers essentiels dans votre projet

   ├── service.py    → /votre/projet/HelpIT/

Étape 2 : Vérifier l'installation
   python verify_installation.py

Étape 3 : Tester
   python main.py
   → Validez une cible
   → Cliquez sur "Processus" ou "Services"


📋 FONCTIONNALITÉS
═══════════════════

ProcessManager (process.py)
───────────────────────────
✓ Affiche tous les processus distants
✓ 1 bouton "Kill" par processus (vert → rouge)
✓ Confirmation avant suppression
✓ Rafraîchissement de la liste

ServiceManager (service.py)
───────────────────────────
✓ Affiche tous les services avec 3 colonnes (id, name, état)
✓ 3 boutons par service (Stop, Run, Restart)
✓ États colorés (RUNNING=vert, STOPPED=rouge, etc.)
✓ Mise à jour automatique après chaque action
✓ Rafraîchissement de la liste
